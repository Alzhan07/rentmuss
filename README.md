# RentMus - Аренда музыкальных пространств

Мобильное приложение для аренды сцен, музыкальных инструментов и студий звукозаписи.

## Особенности

### Дизайн
- 🎨 Современный темный дизайн с музыкальной тематикой
- 🎵 Градиенты в фирменных цветах (темно-синий, красный)
- 📱 Адаптивный UI для мобильных устройств
- ✨ Анимации и плавные переходы

### Экраны

#### 1. Вход (Login)
- Красивая форма входа с градиентным фоном
- Валидация email и пароля
- Показ/скрытие пароля
- Переход на регистрацию

#### 2. Регистрация (Register)
- Полная форма регистрации (имя, email, пароль, подтверждение)
- Валидация всех полей
- Проверка совпадения паролей
- Автоматический переход после успешной регистрации

#### 3. Главный экран (Home)
- **Поиск** по всем категориям
- **Категории**: Сцены, Инструменты, Студии, Оборудование
- **Популярные** предложения
- **Рекомендации** на основе предпочтений
- Карточки с:
  - Фото
  - Рейтингом
  - Ценой
  - Местоположением
  - Кнопкой избранного

#### 4. Профиль (Profile)
- **Аватар пользователя**
  - Загрузка из галереи
  - Предпросмотр
  - Автоматическое сжатие
- **Меню настроек**:
  - Избранное (с счетчиком)
  - Смена пароля (модальное окно)
  - Редактирование профиля
  - Уведомления
  - Помощь и FAQ
- **Кнопка выхода**

#### 5. Избранное (Favorites)
- Список сохраненных площадок/инструментов
- Фильтрация по типу
- Быстрое удаление из избранного
- Переход к бронированию
- Пустое состояние с подсказкой

## API Endpoints

### Аутентификация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `POST /api/auth/logout` - Выход

### Профиль
- `GET /api/user/profile` - Получить профиль
- `PUT /api/user/profile` - Обновить профиль
- `POST /api/user/avatar` - Загрузить аватар
- `POST /api/user/change-password` - Сменить пароль

### Избранное
- `GET /api/user/favorites` - Список избранного
- `POST /api/user/favorites/:id` - Добавить в избранное
- `DELETE /api/user/favorites/:id` - Удалить из избранного

### Площадки
- `GET /api/venues` - Список площадок
- `GET /api/venues/:id` - Детали площадки
- `POST /api/venues/:id/book` - Забронировать

## Установка и запуск

### 1. Установить зависимости
```bash
flutter pub get
```

### 2. Настроить API URL
Откройте `lib/services/api_service.dart` и измените `baseUrl`:
```dart
static const String baseUrl = 'http://your-server:5000/api';
```

### 3. Запустить приложение
```bash
flutter run
```
