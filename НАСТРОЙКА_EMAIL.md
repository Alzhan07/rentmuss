# 📧 Настройка Email для восстановления пароля

## Текущий режим: DEV MODE

Сейчас коды восстановления пароля выводятся в консоль сервера.

Чтобы отправлять письма на **реальный email**, выполните 3 простых шага:

---

## ⚡ Быстрая настройка (5 минут)

### Шаг 1: Создайте пароль приложения Gmail

1. Откройте: https://myaccount.google.com/apppasswords
2. Если требуется - включите 2FA (двухфакторную аутентификацию)
3. Выберите:
   - **App**: Mail
   - **Device**: Other → напишите "RentMus"
4. Нажмите **Generate**
5. **Скопируйте** 16-значный пароль (например: `abcd efgh ijkl mnop`)

### Шаг 2: Обновите файл .env

Откройте файл: `lib/server/.env`

Найдите эти строки:
```env
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```

Замените на свои данные:
```env
EMAIL_USER=ivan.petrov@gmail.com
EMAIL_PASS=abcdefghijklmnop
```

> ⚠️ Используйте **пароль приложения** (16 символов), а не обычный пароль Gmail!

### Шаг 3: Перезапустите сервер

```bash
cd lib/server
node server.js
```

### ✅ Готово!

При запуске вы должны увидеть:
```
📧 Email service configured with SMTP: smtp.gmail.com
```

---

## 🧪 Тестирование

1. Запустите приложение
2. На экране логина нажмите "Забыли пароль?"
3. Введите email
4. Проверьте почту - должно прийти письмо с кодом

---

## 🔍 Решение проблем

### Письма не приходят

1. Проверьте папку "Спам"
2. Убедитесь, что используете **пароль приложения**, а не обычный пароль
3. Проверьте консоль сервера на наличие ошибок

### "Invalid login: 535-5.7.8"

- Включите 2FA в Google аккаунте
- Создайте новый пароль приложения
- Проверьте, что EMAIL_USER содержит полный адрес (с @gmail.com)
